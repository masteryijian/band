# 清新空气乐队网站

“清新空气”乐队的单页官方网站，包含乐队介绍、2026 夏季巡演、成员资料和音乐预告。

## 本地预览

这是一个不依赖构建工具的静态网站。可以直接打开 `index.html`，或者在当前目录启动任意静态文件服务器。

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 内容维护

- 演出日期：修改 `index.html` 中 `data-show-date` 和巡演卡片日期。
- 成员资料：修改 `#members` 区域中的成员卡片。
- 场地与联系方式：当前尚未公布，确认信息后替换页面中的提示文字。
- 主视觉：`band.png`。

## 技术

HTML、CSS 和原生 JavaScript，无第三方依赖。

## 代码约定

HTML、CSS 和 JavaScript 使用简洁的中文结构注释，解释页面分区和关键逻辑。注释重点说明“为什么这样做”，避免逐行复述代码。
