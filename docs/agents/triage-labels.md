# Triage Labels

工程技能使用五个固定角色。下表将这些角色映射到本仓库实际使用的
GitHub 标签。

| Label in skills | Label in this repository | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，可交给 agent 执行 |
| `ready-for-human` | `ready-for-human` | 需要人工实现或决策 |
| `wontfix` | `wontfix` | 不会处理 |

技能提到某个 triage 角色时，使用表格第二列对应的 GitHub 标签。
